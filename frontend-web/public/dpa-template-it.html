2. Descrizione del Trattamento

Dataxiom fornisce il servizio Badge System — piattaforma cloud di tracciamento presenze e gestione turni per dipendenti retail.

Categoria Dati Finalità Durata Conservazione
Identificativi
(nome, email, matricola, telefono)		 Autenticazione, reporting presenze Durata rapporto + 12 mesi

(modifiche, correzioni, azioni admin)		 Traccia compliance, investigazioni 3 anni (obbligo legale)

3. Localizzazione e Infrastruttura

Nessun trasferimento extra-UE. Tutti i dati sono conservati su infrastruttura AWS nella regione eu-west-1 (Irlanda).

  • Database: Amazon RDS PostgreSQL (Irlanda) — encryption at rest AES-256
  • API Backend: Amazon EC2 (Irlanda) — HTTPS TLS 1.3
  • Backup: AWS automated (7-day retention, point-in-time recovery)

4. Obblighi di Dataxiom (Art. 32 GDPR)

  • Controllo d'accesso MFA per admin Dataxiom; access log registrati
  • Crittografia TLS 1.3 in transit, AES-256 at rest; chiavi in AWS Secrets Manager
  • Coordinate GPS cancellate automaticamente dopo 90 giorni
  • Backup automatici giornalieri; disaster recovery RTO 2h
  • Notifica breach: entro 2 ore da scoperta → report scritto al Cliente entro 24 ore
  • Monitoraggio CloudWatch + Sentry + audit log per tutte le modifiche
  • OWASP Top 10 code review pre-deployment; npm audit su ogni deploy
  • Vietato: condividere dati con terzi non autorizzati, usare dati per scopi propri, trasferire extra-UE, usare per training AI senza consenso

5. Obblighi del Cliente (Art. 6 GDPR)

  • Garantire base legale per il trattamento (contratto di lavoro, obbligo legale)
  • Fornire Privacy Policy ai dipendenti PRIMA di usare Badge System
  • Raccogliere consenso esplicito (Art. 9) per Face ID; consenso (Art. 7) per geofencing GPS
  • Notificare Dataxiom entro 24h da qualsiasi breach (credenziali compromesse, accesso non autorizzato)
  • Cooperare con audit su richiesta di Dataxiom

6. Diritti degli Interessati

  • Accesso (Art. 15): Dataxiom fornisce export CSV entro 10 giorni
  • Cancellazione (Art. 17): Dataxiom cancella entro 7 giorni (salvo retention obbligatoria)
  • Rettifica (Art. 16): Manager/Admin corregge via dashboard entro 2 giorni
  • Portabilità (Art. 20): Export CSV/JSON entro 10 giorni

7. Post-Risoluzione Contratto

Entro 30 giorni dalla risoluzione, Dataxiom:

  • Esporta tutti i dati personali in formato CSV/JSON (portabilità Art. 20)
  • Cancella tutti i dati dal database di produzione
  • Fornisce certificato di cancellazione firmato dal CTO Dataxiom
  • Conserva audit log per 7 giorni aggiuntivi (compliance finale)
